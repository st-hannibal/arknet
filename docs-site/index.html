arknet is a decentralized network for AI inference. Run a node with any GPU, serve any open model, earn tokens. Users get an OpenAI-compatible API backed by thousands of nodes instead of one company.

# Run a node
curl -fsSL https://arknet.arkengel.com/install.sh | sh
arknet init --network mainnet
arknet start --role compute
# Use as a developer — change one line from OpenAI
# The network routes your request to the best available node
client = OpenAI(base_url="https://api.arknet.arkengel.com/v1", api_key="unused")
response = client.chat.completions.create(
    model="meta-llama/Llama-3.1-8B-Instruct",
    messages=[{"role": "user", "content": "Hello from arknet"}],
)

Fair launch. No premine. No investors. No team tokens. No airdrop. Every ARK is minted against verified inference work. The maintainer holds zero tokens at genesis.

How it works

Compute nodes

Run AI models on your hardware. Earn 75% of the reward for every verified inference job you serve.

Verifiers

Re-execute 5% of jobs deterministically. Catch cheaters, earn 7% of rewards. Fake output = slashed stake.

Routers

Dispatch user requests to the best compute node. Earn 5% for orchestrating the network.

Validators

Run BFT consensus. Finalize receipts, mint rewards, govern the protocol. Tendermint-based instant finality.

One binary, four roles

Every participant runs the same arknet binary. Configure which roles to enable based on your hardware:

[roles]
validator = false
router    = true
compute   = true     # needs GPU
verifier  = false

A Raspberry Pi can route. A gaming laptop can compute. A server rack can do everything. The network adapts.
