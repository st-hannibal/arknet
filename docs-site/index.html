Run a node (when testnet is live)

The one-liner installs a prebuilt arknet binary from GitHub Releases. Until Phase 1 ships a signed release, build from source.

curl -fsSL https://arknet.arkengel.com/install.sh | sh

Build from source (Phase 0)

git clone --recursive https://github.com/st-hannibal/arknet.git
cd arknet
cargo build --release
./target/release/arknet --help
