What is arknet?

arknet is a peer-to-peer network for AI inference. Anyone with a GPU can earn ARK tokens by serving AI models. Any developer can use AI through a Python SDK that connects directly to the mesh — no API keys, no centralized servers, no HTTP.

The entire network is trustless and encrypted. Your prompts travel over Noise-encrypted P2P channels. Only the compute node running your job sees the prompt — validators relay opaque bytes they cannot read. There is no company in the middle.

Quick start

# Install the SDK and connect to the mesh
pip install arknet-sdk

from arknet_sdk import Wallet, Client

wallet = Wallet.create()
session = wallet.create_session(spending_limit=100_000_000, expires_secs=3600)
client = Client.connect(session=session)
response = client.infer(model="Qwen/Qwen3-0.6B-Q8_0", prompt="Hello!", max_tokens=64)

That is it. The SDK joins the gossip mesh, discovers compute nodes via PoolOffer gossip, and sends your inference request through the network. No URLs, no API keys, no HTTP.

Fair launch. No premine. No investors. No team tokens. No airdrop. Every ARK is minted against verified inference work. The maintainer holds zero tokens at genesis.

How it works


    SDK (your app)
        |
        | 1. joins gossip mesh via seed nodes
        |
    Validator / Seed
        |
        | 2. discovers compute via PoolOffer gossip
        |    (libp2p relay + hole punching)
        |
    Compute Node (GPU)
        |
        | 3. runs inference, streams tokens back
        | 4. receipt submitted to chain
        |
    Validator Consensus (BFT)
        |
        | 5. verifier audits, reward minted
        |
    ARK minted to compute (80%)

Compute nodes

Run AI models on your hardware. Earn 80% of the reward for every verified inference job you serve.

Verifiers

Re-execute 5% of jobs deterministically. Catch cheaters, earn 7% of rewards. Fake output = slashed stake.

Validators

Run BFT consensus, serve as seed/relay nodes. Finalize receipts, mint rewards, govern the protocol. Instant finality.

Delegators

Stake ARK on compute nodes and earn 5% of their rewards pro-rata. No hardware required.

One binary, two node types

Every participant runs the same arknet binary. Configure which role to enable based on your hardware:

[roles]
validator = true    # consensus + seed + relay
compute   = true    # inference + gossip (needs GPU)

A server can validate. A gaming laptop can compute. Both roles can run together on the same machine.

Pure peer-to-peer

There are no HTTP endpoints, no gateways, no middlemen. The SDK joins the libp2p gossip mesh directly, discovers compute nodes through PoolOffer messages, and routes inference via relay + hole punching. All traffic is Noise-encrypted end-to-end. Validators relay encrypted bytes they cannot read.

Session keys

Your wallet creates ephemeral session keys with a spending limit and expiry. Use session keys for daily SDK calls — if one is compromised, the attacker can only spend the session allowance, not drain your wallet.

session = wallet.create_session(
    spending_limit=100_000_000,  # 0.1 ARK
    expires_secs=3600            # 1 hour
)

Fair launch

arknet launches with zero tokens in circulation. No premine. No investors. No team allocation. No airdrop. Every ARK is minted against verified inference work — the same model as early Bitcoin mining.

During a 6-month bootstrap period, inference is free and rate-limited. Compute nodes earn ARK from block emission for serving free requests. By the end of bootstrap, enough ARK circulates for the paid economy to activate naturally.
